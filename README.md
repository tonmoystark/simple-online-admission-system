# 🎓 Online Admission System (PHP Project)

## 📌 Overview

This is a simple and user-friendly **Online Admission System** developed using **Core PHP and MySQL**.
The system allows students to submit admission applications online, and administrators can view and manage those applications from a dashboard.

---

## 🚀 Features

### 👨‍🎓 Student Side

* Fill out admission form
* Submit personal and course details
* Simple and clean interface

### 🛠️ Admin Panel

* View all submitted applications
* Organized table format
* Easy data management

---

## 🧰 Technologies Used

* **Frontend:** HTML, CSS
* **Backend:** PHP
* **Database:** MySQL
* **Server Environment:** Laravel Herd / XAMPP

---

## ⚙️ Installation Guide

1. Clone the repository:

```
git clone https://github.com/your-username/admission-system.git
```

2. Move the project to your local server directory (Herd / htdocs)

3. Create a database in phpMyAdmin:

```
admission_system
```

4. Import the `database.sql` file

5. Run the project:

```
http://admission-system.test
```
